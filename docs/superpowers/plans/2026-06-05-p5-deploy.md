# P5: Deploy Implementation Plan (PM2)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deploy the Next.js app to vps6core at `preview.tdgamestudio.com` using PM2 + nginx with SSL.

**Architecture:** On the VPS, the repo is cloned to `/opt/tdgames-preview`. `npm run build` produces the `.next` build. PM2 runs `next start` on port 3001, keeping the process alive and auto-restarting on crash/reboot. Nginx reverse-proxies `preview.tdgamestudio.com → 127.0.0.1:3001` with SSL via Certbot. Future redeploys: `./scripts/deploy-vps.sh` from local machine.

**Tech Stack:** PM2 6.0.14 (already on VPS), Node.js 22 (already on VPS, compatible with Next.js 14), nginx 1.18, Certbot (Let's Encrypt). Access VPS via `tailscale ssh root@vps6core`.

## Task 2: Deploy Script

**Files:**
- Create: `scripts/deploy-vps.sh`

Run this locally to push any branch to production. Combines `git pull + npm ci + npm run build + pm2 restart` in one SSH call.

- [ ] **Step 1: Create scripts/ directory and deploy script**

```bash
mkdir -p scripts
```

```bash
#!/usr/bin/env bash
# scripts/deploy-vps.sh
# Usage: ./scripts/deploy-vps.sh [branch]
# Default branch: feat/p1-foundation-auth
set -euo pipefail

VPS="root@vps6core"
APP_DIR="/opt/tdgames-preview"
BRANCH="${1:-feat/p1-foundation-auth}"

echo "==> Deploying branch '$BRANCH' to vps6core..."

ssh "$VPS" bash << REMOTE
set -euo pipefail
cd $APP_DIR

echo "--- git pull ---"
git fetch origin
git checkout $BRANCH
git pull origin $BRANCH

echo "--- npm ci ---"
npm ci

echo "--- npm run build ---"
npm run build

echo "--- pm2 restart ---"
pm2 restart tdgames-preview 2>/dev/null || pm2 start ecosystem.config.js
pm2 save

echo "--- status ---"
pm2 show tdgames-preview | grep -E 'status|uptime|memory|pid'
REMOTE

echo "==> Done. App running at https://preview.tdgamestudio.com"
```

- [ ] **Step 2: Make executable + commit + push**

```bash
chmod +x scripts/deploy-vps.sh
git add scripts/deploy-vps.sh
git commit -m "feat(p5): add VPS deploy script (git pull + npm build + pm2 restart)"
git push origin feat/p1-foundation-auth
```

## Task 4: nginx Config + SSL

Still on VPS (`tailscale ssh root@vps6core`).

> **Prerequisite:** DNS A record `preview.tdgamestudio.com` must point to VPS public IP.  
> Run `curl -s ifconfig.me` on VPS to get the public IP, then add an A record at your DNS provider.

- [ ] **Step 1: Get VPS public IP**

```bash
curl -s ifconfig.me
```

Add A record: `preview.tdgamestudio.com → <that IP>`. Wait for DNS propagation (usually 1–5 min with short TTL).

- [ ] **Step 2: Create nginx config (HTTP first, for Certbot)**

```bash
cat > /etc/nginx/sites-available/preview.tdgamestudio.com << 'EOF'
server {
    listen 80;
    server_name preview.tdgamestudio.com;

    # Large body for asset uploads
    client_max_body_size 100M;

    location / {
        proxy_pass http://127.0.0.1:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_read_timeout 60s;
    }
}
EOF
```

- [ ] **Step 3: Enable site + reload nginx**

```bash
ln -s /etc/nginx/sites-available/preview.tdgamestudio.com /etc/nginx/sites-enabled/
nginx -t
systemctl reload nginx   # reload, not restart — keeps other 11 apps alive
```

Expected: `syntax is ok` + `test is successful`.

Test HTTP:
```bash
curl -s -o /dev/null -w "%{http_code}" http://preview.tdgamestudio.com/
```

Expected: `307`.

- [ ] **Step 4: SSL via Certbot**

```bash
certbot --nginx -d preview.tdgamestudio.com
```

Certbot will:
1. Ask for email + ToS agreement
2. Fetch Let's Encrypt cert
3. Auto-modify nginx config to add HTTPS + redirect HTTP→HTTPS

Expected final message: `Successfully deployed certificate for preview.tdgamestudio.com`.

- [ ] **Step 5: Verify HTTPS**

```bash
curl -s -o /dev/null -w "%{http_code}" https://preview.tdgamestudio.com/
```

Expected: `307`.

Open in browser: `https://preview.tdgamestudio.com` → shows login page with valid SSL lock.
